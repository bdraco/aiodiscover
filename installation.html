Installation

Stable release

To install Async Host discovery, run this command in your terminal:

$ pip install aiodiscover

This is the preferred method to install Async Host discovery, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources

The sources for Async Host discovery can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/bdraco/aiodiscover

Or download the tarball:

$ curl  -OL https://github.com/bdraco/aiodiscover/tarball/main

Once you have a copy of the source, you can install it with:

$ python setup.py install
